# SEO Standards

## Metadata

Every public page must define:

* Title (50–60 characters)
* Meta description (140–160 characters)
* Open Graph metadata
* Twitter card metadata

## App Router Metadata

* Use `export const metadata = {}`
* No client-side metadata generation

## Semantic HTML

* One H1 per page
* Logical heading order
* Proper landmark elements

## Images

* Required alt text
* Descriptive and keyword-aware

## Indexing

* No accidental noindex
* Crawlable navigation
* SSR for SEO-critical pages
